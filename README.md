# Petunjuk Deploy

Disini saya menggunakan Framework Laravel 7 dengan database menggunakan php MySql

## Installation

1. Clone Repository terlebih dahulu
2. Kemudian ketikan

```bash
cp .env.example .env
```
3. Kemudian ketikan
```bash
php artisan key:generate
```
4. Kemudian ketikan
```bash
php artisan serve
```
